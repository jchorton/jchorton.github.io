# About

This is the personal website of *Jonathan Horton*, eternal student in the realm of computing.

Hosting is done for free using GitHub Pages. The [source code](https://github.com/jchorton/jchorton.github.io) is publicly available.

There are excellent guides for setting this up [here](http://jmcglone.com/guides/github-pages/) and a second [here](https://readwrite.com/2013/11/27/github-pages-explained/). The pages use [kramdown](https://kramdown.gettalong.org/quickref.html) as the Markdown interpretter.
